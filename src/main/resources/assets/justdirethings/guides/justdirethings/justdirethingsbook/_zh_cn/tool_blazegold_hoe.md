# 烈焰金锄

[核源铁锄](./tool_ferricore_hoe.md)升级而来，挥舞速度更快，且拥有“熔岩修补”。用其耕犁泥土等可将其变为烈焰蕾壤；这种土壤助农作物生长的能力更强，还具有自动收获功能——其上植物会自动掉落资源。

收获到的物品很快就会消失。需使用[物品拾取器](./mach_itemcollector.md)高效收集掉落物。
