# 蚀空燃液

蚀空燃液的原料是未精炼的蚀空燃液。第一步需向[虚炎燃液](./res_refined_fuel_t3.md)中投入[蚀空烬质](./res_coal_t4.md)。再在暗影脉凝胶旁边倒出所得流体即可。这种燃液的品质最高，每刻产能速度和总产能也最高，是高级机器与设施的理想能源物质。

蚀空燃液是一种高效的燃料，可在[燃液发电器](./mach_generatorfluidt1.md)中用于产生能量。
