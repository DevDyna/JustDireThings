# 虚炎燃液

虚炎燃液由未精炼的虚炎燃液加工而来。首先，向[烈焰烬质](./res_refined_fuel_t2.md)中投入[虚炎煤炭](./res_coal_t3.md)。然后，令所得流体与虚空辉凝胶接触，进行精炼。此燃液的能量产出高于前代燃液，每刻产能速度和总产能都得到了大幅提升。

虚炎燃液是一种高效的燃料，可在[燃液发电器](./mach_generatorfluidt1.md)中用于产生能量。
