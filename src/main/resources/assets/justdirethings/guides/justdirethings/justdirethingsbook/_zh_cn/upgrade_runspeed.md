# 疾跑速度

疾跑速度升级能强化你的疾跑能力。此升级会大幅加快你疾跑的速度，长途旅行便不再费时。

可以加装于核源铁护腿，更高品质护腿的疾跑速度也更快。设置界面中有滑块可供调整。
