# 烧灼清创

烧灼清创升级是提高自身幸存概率的不二之选。此升级能让你在受到伤害后快速恢复生命值，它会借用战斗所生的热量来烧灼封闭创口。

右击可激活此能力；它会立即治疗你，并消耗一些耐久度。需注意，此能力使用之后会进入冷却。
