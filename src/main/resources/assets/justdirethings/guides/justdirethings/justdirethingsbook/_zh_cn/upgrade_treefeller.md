# 伐木

伐木升级无需激活，适用于核源铁及以上的斧。破坏木质方块时，它可让所有相连的同类木质方块同时被破坏。

一挥便可砍倒整棵树！

注意：砍伐整树的耗时略长于破坏单个方块！
