# 高级方块放置器

高级方块放置器是[初级方块放置器](./mach_blockplacert1.md)的升级版。

过滤器限定了何种方块可以作为放置的基础。比如，过滤石头，则放置器只能将方块等放置到石头上。过滤效果也会受到方向设置的影响；方向设定为“下”，其余与先前一致，则其只会放置到石头的**顶面**上。
