# 垂直清扫

垂直清扫升级无需激活，适用于核源铁及以上的锹。当破坏受重力影响的方块（比如沙砾和沙子），其上方所有同受重力影响的方块均会自动被破坏。挖矿时再也不必担心沙砾塌方了！
