# 核源铁锄

核源铁锄是开垦与耕种的绝佳工具。它接受多种升级。

用于开垦泥土时，核源铁锄会将其转变为原初浆壤。这种土壤不会被踩坏，可保证其上农作物完好无损。这些农作物的生长速度也略有加快。
