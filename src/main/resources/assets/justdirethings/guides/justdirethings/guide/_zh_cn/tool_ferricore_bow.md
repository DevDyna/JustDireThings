# 核源铁弓

核源铁弓，以坚韧的核源铁锻成，以其较高的耐久度强化了普通弓的能力，还可加装各式升级。

此弓的工具设置界面中有一个特殊的药水罐槽，可赋予箭矢多种药水效果。放入[药水罐](./item_potion_canister.md)即可。
