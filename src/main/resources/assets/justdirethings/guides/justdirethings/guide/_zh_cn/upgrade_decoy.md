# 诱饵

诱饵升级能让你的敌人分神。部署出的诱饵会吸引敌人的注意力，从而给予你战略优势。

用设定键位激活此能力，即会生成一个诱饵。其会在一段时间内强制附近的敌人攻击它，直至自身消失。
