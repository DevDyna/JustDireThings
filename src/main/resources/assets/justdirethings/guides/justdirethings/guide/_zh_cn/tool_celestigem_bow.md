# 苍穹晶弓

苍穹晶弓超乎[烈焰金弓](./tool_blazegold_bow.md)之上，不再使用耐久度，而是换用 FE。它最多能容纳 10,000FE，足够长时间使用。

此弓有 3 个“药水罐槽”，可供附加状态效果。放入[药水罐](./item_potion_canister.md)即可加强箭矢。
