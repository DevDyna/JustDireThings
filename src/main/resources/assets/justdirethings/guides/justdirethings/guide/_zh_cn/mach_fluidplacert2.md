# 高级流体放置器

高级流体放置器由[初级流体放置器](./mach_fluidplacert1.md)升级而来。

它能在更大的区域内放置流体。和[高级方块放置器](./mach_blockplacert2.md)类似，它的过滤槽也能选定何种方块上可以放置流体。
