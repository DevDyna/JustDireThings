# 抵消摔落

抵消摔落升级能避免意外摔落造成伤害。此升级会完全吸收摔落伤害，着陆时便不再有受伤之虞。

注意：缓冲摔落需要消耗能量。抵消的伤害越多，消耗的能量也越多。苍穹晶靴子可没有那么多能量……
