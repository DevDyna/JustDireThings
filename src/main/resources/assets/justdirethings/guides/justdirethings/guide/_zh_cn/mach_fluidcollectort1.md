# 初级流体收集器

初级流体收集器可以自动盛取世界中的流体。它会尝试收集正前方的流体，并存入其内部的存储空间。需注意它只能同时存储 1 桶流体。

物品槽可放置空铁桶以盛装流体，也接受流体罐等可以盛装流体的物品。
