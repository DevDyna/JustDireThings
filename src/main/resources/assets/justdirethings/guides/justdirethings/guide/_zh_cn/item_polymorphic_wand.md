# 多态化形手杖

多态化形手杖利用了烈焰金蕴含的惰性能量，以激活多态流体，并将生物转换成另一种生物！

只需对敌对生物激活手杖，它就将变成另一种敌对生物。对友好生物的效果一致。

构造如此简单的法杖不能保证转换结果可控，也无法相互转换敌对和友好生物。即便如此，你也认为这杆手杖可以升级，从而能完成这些任务！
