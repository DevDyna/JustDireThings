# 跳跃提升

跳跃提升升级能让你跳得更高。此升级会增加你的跳跃高度，跨越障碍、攀爬高峰便不是难事。

可以加装于核源铁靴子，更高品质靴子的跳跃高度也更高。设置界面中有滑块可供调整。
