# 生物捕捉器

生物捕捉器是能捕捉和释放实体的新颖工具。对着实体掷出即可捕捉该实体，捕捉后再掷出即可释放。

也可以放到某些过滤器中，用以过滤特定类型的实体，比如蓝色的绵羊。
