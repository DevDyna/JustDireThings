# 穿墙

穿墙升级可令你穿过墙壁与障碍。激活即可消去自身的实体，以此无视所有物理障碍，地面除外。

激活时就可穿透墙壁。某些方块依然无法穿过，比如基岩，以及那些整合包作者指定的方块。
