# 暗影脉凝胶

暗影脉凝胶是凝胶研究的集大成者，它能将下界合金转化成蚀空合金。这种顶级凝胶可用于制造极其强力的物品。

暗影脉凝胶的合成需要用到最稀有的和最强大的材料，它在物质转化上的能力也因此无与伦比。
