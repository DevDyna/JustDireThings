# 苍穹晶锄

由[烈焰金锄](./tool_blazegold_hoe.md)升级而来，苍穹晶使用 FE，而非耐久度。此锄能开垦出虚空辉壤。

如果锄上加装有“掉落物传送”能力，且已绑定至某容器，则开垦出的耕地会将自动收获的产物存入其中。
